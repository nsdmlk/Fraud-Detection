# Fraud Detection

Binary classification model for detecting fraudulent credit card transactions.

## Dataset

[Credit Card Fraud Detection](https://www.kaggle.com/datasets/mlg-ulb/creditcardfraud) — 284,807 transactions, 492 frauds (0.17%).

## Approach

- SMOTE for handling class imbalance
- Random Forest & XGBoost classifiers
- Evaluation: precision, recall, F1-score, confusion matrix

## Tech Stack

Python — pandas, scikit-learn, XGBoost, imbalanced-learn
